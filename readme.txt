This is a simple plugin to insert Ads to your website. As the name says you can show random Ads within your post content.

== Description ==

Inserting Ads to your website is made simple by **Random Ads by Aarvis**. Using this plugin you can insert maximum 6 Ad blocks into your post content.
Inserting Ads manually to every post is a tedious work. But now you can achieve it with one time setting with the help of **Random Ads by Aarvis**. 

= Main Features =

* Simple and user friendly interface.
* No programming knowledge required to implement.
* Insert Ads into yor post content automatically. One time setting is required.
* Inserts Ads after every specific number of paragraphs.
* Inserts Ads after every predefined number of characters in the post.
* Set the max number of Ads to be shown in every post.
* Selects random Ads from a collection of ad blocks. If this option is unchecked, Ads will be shown in the existing order.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/random-ads-by-aarvis` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the **Random Ads by Aarvis** link to configure the plugin.
